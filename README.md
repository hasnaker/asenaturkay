# asenaturkay.com

Doktorant Psikolojik Danışman Asena Türkay'ın kişisel web sitesi.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS (soft pastel tema)
- AWS Amplify Hosting

## Geliştirme
```bash
npm install
npm run dev
```

## Deploy
GitHub'a push → AWS Amplify otomatik build & deploy.
